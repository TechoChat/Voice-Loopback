# voice_loopback

A new Flutter project.

## Getting Started

This project aims to provide an wireless mic function using your phones microphone and bluetooth/wiredheadset as a speaker.

## Features

- Wireless mic function
- Bluetooth/wiredheadset support
- Audio processing
- Audio visualization

## Installation

- Flutter
- Android Studio
- Android SDK
- Android NDK
- Android Emulator
- Android Device

## Usage

- Run the app on your Android device
- Connect your Bluetooth/wiredheadset
- Start the app
- Start the mic
- Start the speaker
